Basic carousel

This example shows how to setup a basic carousel with prev/next controls, pagination, swipe support and lazy loading.

‹ ›

Init:
$('.jcarousel-1')
    .jcarousel()
    .jcarouselSwipe()
    .jcarouselLazyLoading();
‹ ›

Init:
$('.jcarousel-2')
    .jcarousel()
    .jcarouselSwipe()
    .jcarouselLazyLoading({
        preventScroll: false
    });
