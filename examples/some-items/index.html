Carousel with some items

This example shows how to setup a carousel with some items with prev/next controls, swipe support and lazy loading.

‹ ›

Init:
$('.jcarousel-1')
    .jcarousel({
        wrap: 'circular'
    })
    .jcarouselSwipe()
    .jcarouselLazyLoading();
‹ ›

Init:
$('.jcarousel-2')
    .jcarousel({
        wrap: 'circular'
    })
    .jcarouselSwipe()
    .jcarouselLazyLoading({
        preventScroll: false
    });
